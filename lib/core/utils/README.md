# Utils Organization

This directory contains utility classes that provide helper functions for various aspects of the application.

## File Organization

- **color_utils.dart**: Color-related utilities (conversion, generation, etc.)
- **date_time_utils.dart**: Date and time manipulation utilities
- **dialog_utils.dart**: Dialog creation and management utilities
- **file_utils.dart**: File operations utilities
- **format_utils.dart**: Formatting utilities for numbers, text, etc.
- **log_utils.dart**: Logging utilities
- **navigation_utils.dart**: Navigation and routing utilities
- **network_utils.dart**: Network-related utilities
- **permission_utils.dart**: Permission handling utilities
- **string_utils.dart**: String manipulation utilities
- **ui_utils.dart**: UI-related utilities
- **validation_utils.dart**: Input validation utilities

## Guidelines

1. Each utility class should focus on a specific domain of functionality
2. Methods should be static and stateless when possible
3. Avoid dependencies on repositories or services
4. Keep methods small and focused on a single task
5. Use descriptive method names that clearly indicate what the method does
